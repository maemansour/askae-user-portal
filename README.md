The **Askae User Portal** is a front-end dashboard built with Next.js (App Router) for onboarded users of **Askae**, a platform that helps small wellness businesses automate their phone calls with AI.

This dashboard provides business owners with an intuitive interface to:
- Review AI-handled calls and track call activity  
- Monitor call status, performance, and insights  
- Create new AI phone agents  
- Manage existing agents and configurations  
- View payment history and subscription details  
- Access account and usage information  

The focus of this project is on the **front-end experience**:  
designing a clean, responsive dashboard UI, building reusable components, organizing navigation and layouts, and integrating with backend APIs to display real user data.
